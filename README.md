# Angular

Requerimientos:
- JDK
- IDE (Net Beans, Eclipse u otro)
- Servidor Apache Tomcat
- Node.js
- NPM

Instrucciones para ejecutar el proyecto:

- Debemos ejecutar el proyecto API_Rest_Spring_Boot en un servidor Apache TOMCAT y puerto por defecto (8080).
